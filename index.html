This is a serverless contact form built using AWS.

How it works:

  1. The user fills out the form and hits send.
  2. The users browser sends an HTTP POST request with the form data in the request body to an API endpoint specified by AWS API Gateway.
  3. When the API endpoint receives the HTTP request, it validates the request and then passes the request body to a Lambda function.
  4. The Lambda function extracts the form data from the request body then it uses that data to build the body of the email.
  5. The Lambda function then invokes AWS Send Email Service (SES) passing it the email data.
  6. Finally, SES takes the email data and turns it into an actual email that it sends through AWS mail servers.

Side note: This form uses a honeypot for spam protection.
